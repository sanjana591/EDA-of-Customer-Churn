# 📞 Telecom Customer Churn Analysis (EDA)

## 📌 Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a telecom customer dataset to understand patterns, trends, and factors influencing customer churn. The goal is to extract meaningful insights that can help improve customer retention strategies.

---

## 🎯 Objective
- Analyze customer behavior and service usage  
- Identify key factors affecting churn  
- Discover trends and patterns in customer data  
- Support business decisions with data insights  

---

## 📊 Dataset Description
The dataset includes:
- Customer demographics (Gender, Senior Citizen)  
- Account details (Tenure, Contract Type)  
- Services (Internet, Phone, Streaming)  
- Billing information (Monthly Charges, Total Charges)  
- Churn (Target Variable: Yes/No)  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## 🔍 EDA Workflow
1. Data Loading  
2. Data Cleaning  
   - Handling missing values  
   - Fixing data types  
3. Data Exploration  
4. Univariate Analysis  
5. Bivariate Analysis  
6. Outlier Detection  
7. Correlation Analysis  

---

## 📈 Key Insights
- Customers with **month-to-month contracts** show higher churn rates  
- **High monthly charges** are associated with increased churn  
- Customers with **longer tenure** are less likely to churn  
- Lack of services like **tech support and online security** increases churn probability  
- Senior citizens tend to churn more compared to other groups  

---

## 📊 Visualizations
- Count plots (Churn distribution)  
- Bar charts (Contract vs Churn)  
- Boxplots (Charges vs Churn)  
- Heatmap (Feature correlations)
